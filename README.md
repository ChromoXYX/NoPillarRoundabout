# No Pillar Roundabout

A Cities: Skylines 2 Mod.

Disables roundabout update caused by bridge pillars hovering over intersections,
by removing composition flag `Roundabout` from pillar prefabs.

## Support

Bug reports and feature requests can be made on the [GitHub Issues](https://github.com/ChromoXYX/NoPillarRoundabout/issues),
or sent to [me@chromo.dev](mailto://me@chromo.dev).

## Usage

```bash
dotnet restore
dotnet build
```

DLL binaries will be copied to `AppData\LocalLow\Colossal Order\Cities Skylines II\Mods` automatically.
